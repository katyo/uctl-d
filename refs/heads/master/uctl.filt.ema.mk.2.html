Parameters

n N

The number of samples in range 1.0 ... ∞

Usually the N can be treated as the number of samples for smoothing. This means that when the N equals to 1 then no smoothing will be applied. The more N does more smoothing and vise versa.

α = \frac{2}{n + 1}

Filter formula: y = \frac{2}{n + 1} x + \frac{n - 1}{n + 1} y_{z^{-1}}

Examples

Params from samples

static immutable auto param = mk!Samples(2.0);

assert_eq(param.alpha, 0.6666667, 1e-6);
assert_eq(param.cmpl_alpha, 0.3333333, 1e-6);

Params from samples (fixed)

static immutable auto param = mk!Samples(asfix!2.0);

assert_eq(param.alpha, asfix!0.6666666665);
assert_eq(param.cmpl_alpha, asfix!0.3333333335);
