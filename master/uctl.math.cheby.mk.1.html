uctl math cheby

Generate coefficients for function approximation by fitting Chebyshev polynom in range

  1. template mk(alias C, alias F, real a, real b, uint D)
    template mk (
    alias C
    alias F
    real a
    real b
    uint D
    ) if (
    __traits(isSame, C, Cheby) &&
    isCallable!F
    &&
    Parameters!F.length == 1
    &&
    isNumer!(Parameters!F[0])
    &&
    isNumer!(ReturnType!F)
    ) {
    enum mk;
    }
  2. template mk(alias C, real a, real b, T, alias c)
  3. template mk(alias C, alias F, real a, real b, uint D, T)

Examples

Test Chebyshev sin fit coefficients (floating-point, 2rd)

import std.math: sin;

enum xsin = mk!(Cheby, sin, 0, PI*0.5, 2);

assert_eq(cast(float) xsin[0], 0.602201760f);
assert_eq(cast(float) xsin[1], 0.513518274f);
assert_eq(cast(float) xsin[2], -0.104905032f);
