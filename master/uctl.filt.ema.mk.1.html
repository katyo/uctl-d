Parameters

alpha A

The value of α in range 0.0 .. 1.0

Usually the alpha factor can be treated as the weight of actual value in result. This meaning that when the alpha equals to 1 then no smoothing will be applied. The less alpha does more smoothing and vise versa.

Filter formula: y = α x + (1 - α) y_{z^{-1}}

Test params from alpha

static immutable auto param = mk!Alpha(0.6);

assert_eq(param.alpha, 0.6, 1e-6);
assert_eq(param.cmpl_alpha, 0.4, 1e-6);
