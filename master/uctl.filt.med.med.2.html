med

uctl filt med

Apply median filter

Apply filter to static array values using value picker

  1. auto med(T[N] data)
  2. auto med(T[N] data)
    auto
    med
    (
    alias P
    alias S = bubble_sort
    uint M = 1
    uint N
    T
    )
    (
    ref T[N] data
    )
    if (
    isPicker!(P, T) &&
    isMutable!(ReturnType!P)
    &&
    isNumer!(ReturnType!P)
    &&
    M >= 1
    &&
    M <= N
    )

Parameters

P

element picker function

S

sorting function (uctl.util.sort.bubble_sort by default)

M

number of elements to select (should be 1..N, 1 by default)

N

number of all elements

T

element type

data T[N]

array of elements

Return Value

Type: auto

Filtered value

To get more smooth you should select lesser M and vice versa.

When M == N than behavior of this filter is same as average filter.

Note: This function modifies data.

Examples

Test median filter with picker

struct A {
  bool b;
  float a;
}

static auto a = [A(true, 0.5), A(true, 0.0), A(false, 0.0625), A(false, 3.15), A(true, -0.25)].staticArray;

assert_eq(a.med!(ref (ref A a) => a.a), 0.0625);
